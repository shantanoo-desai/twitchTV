# twitchTV
[Twitch.tv](https://twitch.tv) Front End for FreeCodeCamp

## Usage

use it with `virtualenv` from python on your machine

    git clone https://github.com/shantanoo-desai/twitchTV.git
    cd twitchTV && virtualenv twitchApp
    pip install -r requirements.txt
    ./run.py


App is hosted on `localhost` on port __30005__

    http://localhost:30005
